Alpheios

Alpheios provides a set of tools for reading classical literatures, and particularly an analyzer of Ancient Greek morphology. This service is based on a modified Morpheus engine originally developed for Perseus Digital Library.

This dictionary was superseded with off-line Morphologia Graeca and will not be updated in the future.

Download

Download the most recent version and add file index.html to GoldenDict following the guide for HTML dictionaries.

Exemplum

States and limitations

In contrast to Eulexis, this dictionary only accepts words written in correct Polytonic orthography with all necessary tones; letter case is significant. So, it will recognize “φιλέω” but not “φιλεω”.
